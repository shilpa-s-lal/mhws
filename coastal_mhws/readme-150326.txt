Coastal Marine Heatwaves (MHWs) in the South West Tropical Pacific (SWTP)
=============


Description
-----------
This dataset contains information on all coastal MHWs for 12 Pacific Island Countries and Territories (PICTs) in the SWTP.
MHWs were detected using the Hobday et al.(2016) method with a 90th percentile threshold and using OSTIA temperature product for the period 1993 to 2023 with the 1993 to 2019 period used as a baseline.

The 12 PICTs are as follows:
New Caledonia labelled as "newcal" in the filenames.
Solomon Islands labelled as "solo" in the filenames.
Vanuatu 
Fiji 
Tonga
Wallis and Futuna labelled as "wf" in the filenames.
Samoa
American Samoa labelled as "amsam" in the filenames.
Niue
Tokelau
Tuvalu
Cook Islands labelled as "cooks" in the filenames.

Files with filename "{countryname}_coastalpoints.csv"provides information on all points along the coastlines of the 12 PICTs where MHW detection was conducted.
These points were identified as best locations for MHW detection as they were located further away from land (and land based contaminants in the air) in waters deeper than 200m based on the GLORYS12 reanalysis product. 
Temperature at these coastal points were extracted from Copernicus. This was done in 2 parts as the temperature product were available in 2 parts with slightly different filenames;  Dataset_id = "METOFFICE-GLO-SST-L4-REP-OBS-SST", for the 1993 to 2007 period and Dataset_id = "METOFFICE-GLO-SST-L4-NRT-OBS-SST-V2" for the period 2007 to 2023. The script is given below. These files were later concatenated to have continuous temperature timeseries from 1993 to 2023 for MHW detection.

import pandas as pd

for i in range (702):  # 702 is the number of coastal points for coastal Fiji

    print(i)

    print ('lat1 = ',file1.latitude[1],'lat2 =',file1.longitude[1])

    file1_ = pd.read_csv(f'/media/shilpa/Expansion/ostia_fiji/fiji_coastal{i}_secondpart.csv')
    print ('lat2 = ',file1_.latitude[1],'lat2 =',file1_.longitude[1])

# Concatenate the files along the columns
    df2_no_header = file1_.reset_index(drop=True)
    combined = pd.concat([file1, df2_no_header], axis=0, ignore_index=True)

# Ensure the 'time' column is in datetime format
    combined['time'] = pd.to_datetime(combined['time'])

# Filter rows to keep only the dates between 1993-01-01 and 2024-10-01
    start_date = '1993-01-01'
    end_date = '2023-10-24'
    filtered = combined[(combined['time'] >= start_date) & (combined['time'] <= end_date)]
    filtered['sst_degC'] = (combined.analysed_sst) - 273.15

# Save the result to a new CSV file
    filtered.to_csv(f'/media/shilpa/Expansion/ostia_full_fiji/fiji_93_23_{i}.csv', index=False)

Requirements
------------
The scripts are using Python and can be run in an Anaconda environment.

Input Data
----------
Temperature timeseries at each coastal point. 

Output
------
There are 2 types of output files:
1. {country_name}_all_mhws_at_{target_latitude}_{target_longitude}.csv

2. {country_name}_mean_trends_mhws_at_{target_latitude}_{target_longitude}.csv

The first type of files contains information on the properties of each MHW at each coastal point. 
Refer to Marine heatwaves package documents developed by Eric Oliver to understand the calculation of each MHW property (https://github.com/ecjoliver/marineHeatWaves).
The second type of file contains trends, means and standard deviation of MHW properties at each coastal point. 


These files can be easily saved into a single dataframe. Here is an example with Fiji data.
__________________________________________________________________________________________

directory = '/media/shilpa/Expansion/ostia_mhw_fiji'
coastal_files_ = [file for file in os.listdir(directory) if file.startswith("Fiji_mean")]

# Read all CSV files and concatenate into a single DataFrame
dfs_ = []
for file in coastal_files_:
    # Handle file names with spaces by adding double quotes around the file path
    file_path = os.path.join(directory, file)
    df = pd.read_csv(file_path)
    dfs_.append(df)

# Concatenate all DataFrames
concatenated_dfs_= pd.concat(dfs_, ignore_index=True)

float_cols__ = concatenated_dfs_.select_dtypes(include='float').columns.difference(['latitude', 'longitude'])

# Round those float columns to 3 decimals
concatenated_dfs_[float_cols__] = concatenated_dfs_[float_cols__].round(3)

concatenated_dfs_.to_csv('/media/shilpa/Expansion/ostia_coastal_mhws_dataverse/means_trends_all_location_country_coastal/fiji.csv')

--------------------------------------------------------------------------------------------------------------------------------------------
In summary, the main output of the "{country_name}_all_mhws_at_{target_latitude}_{target_longitude}.csv" files are:


•country - country of coastal MHW detection
•latitude,longitude - location of coastal point
•index_start,index_end,index_peak - timestep when MHW started, ended and peaked
•date_start,date_end,date_peak - date of MHW start, end and peak
•duration - duration of each MHW event (days)
•intensity_max,intensity_mean,intensity_cumulative - maximum intensity (deg. C), mean intensity (deg. C) and cummulative intensity (deg. C x days) of each MHW
•category - measure of severity of event
•rateonset,ratedecline - onset and decline rate of MHW event (deg. C / days)
•reaction_window_days,coping_window_days - days from start of event to peak of MHW event and days from peak of MHW event to end of the event.



In summary, the main output of the "{country_name}_mean_trends_mhws_at_{target_latitude}_{target_longitude}.csv" files are:


•country - country of coastal MHW detection
•latitude,longitude - location of coastal point
•mean_duration,std_duration - mean and standard deviation of MHW duration at that coastal point
•mean_intensity_mean,std_intensity_mean - mean and standard deviation of MHW intensity mean at that coastal point
•mean_intensity_var,std_intensity_var - mean and standard deviation of MHW intensity variability at that coastal point
•mean_intensity_max,std_intensity_max - mean and standard deviation of MHW intensity max at that coastal point
•mean_intensity_cumulative,std_intensity_cumulative - mean and standard deviation of MHW intensity cumulative at that coastal point
•mean_rate_onset,std_rate_onset - mean and standard deviation of MHW rate onset at that coastal point
•mean_rate_decline,std_rate_decline - mean and standard deviation of MHW rate decline at that coastal point
•mean_total_days,std_total_days - mean and standard deviation of MHW days each year.
•mean_total_icum,std_total_icum - mean and standard deviation of total cumulative intensity of all events each year.
•sig_trend_per_decade_count - significance and trend of number of events per year
