This folder contains all MHWs detected along New Caledonia coastline. As well as trends and means of MHW properties.

All coastal_newcal files are present at : https://drive.google.com/drive/folders/1dL5jI11pVTaYrJceq5Ch-QeIwaOpsNf1?usp=drive_link
